# Decision Record Template

Use this template to document significant technical or process decisions.

---

```markdown
# DR-[NUMBER]: [Decision Title]

**Date:** YYYY-MM-DD
**Status:** Proposed | Accepted | Deprecated | Superseded
**Owner:** [Name]

## Context

What is the situation that requires a decision? What problem are we solving?

## Options considered

### Option A: [Name]

Description, trade-offs, and implications.

### Option B: [Name]

Description, trade-offs, and implications.

## Decision

Which option was chosen and why.

## Consequences

What changes as a result of this decision.
Include both positive outcomes and trade-offs accepted.

## References

Links to related documents, discussions, or external resources.
```
