# Diagrams

This folder contains visual diagrams used across the documentation. All diagrams are written in Mermaid format (`.mmd` files).

## How to view

Mermaid diagrams render natively on GitHub. You can also use:

- The Mermaid Live Editor — paste the file contents to preview
- VS Code with a Mermaid preview extension
- Any Markdown renderer with Mermaid support
